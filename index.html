Sobre el Proyecto

El proyecto consiste en el desarrollo de una aplicación web de código abierto que integra herramientas de auditoría de seguridad y análisis forense digital. La plataforma está diseñada específicamente para pequeñas y medianas empresas (PyMEs) dominicanas que carecen de recursos económicos para adquirir soluciones comerciales de ciberseguridad, cuyo costo promedio oscila entre $50,000 y $200,000 USD anuales. El sistema incluirá tres módulos principales: Port Scanner para identificar vulnerabilidades en la infraestructura de red, Log Analyzer para detectar amenazas web en tiempo real mediante análisis , y Report Generator para consolidar hallazgos en formatos certificables y admisibles como evidencia forense.
